# Bittle X Robot Simulation

Simulation of Petoi's [Bittle X Robot Dog](https://www.petoi.com/products/petoi-robot-dog-bittle-x-voice-controlled).

You can try out the robot before owning it, 
  or without it by your side.

You can also make changes to the onboard firmware, 
  and see the results immediately and safely.

If you are so venturous, you can even code up your
  own robot behaviors from scratch, try it out, 
  and upload to the real robot once you are satisfied!
